## 🚀 Getting Started

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Configure environment
cp .env.example .env  # Edit with your settings

# 3. Start services
docker-compose up -d

# 4. Run the system
python -m cli.chat
```
